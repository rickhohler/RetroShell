# Configuration Files

This directory contains configuration files and templates for your project.

## What goes here?

- Build configuration templates
- Deployment configuration templates
- CI/CD configuration templates
- Environment configuration templates
- Framework or tool-specific configuration templates
- Any other configuration files that should remain private

## Security Note

⚠️ **Never commit sensitive information** like:
- Private keys
- API keys
- Passwords
- Certificates
- Tokens or secrets
- Credentials

Use environment variables or secure storage for sensitive data. This directory should contain **templates only**, not actual secrets or sensitive configuration.
